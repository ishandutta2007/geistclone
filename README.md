<br/>

## Quick Start

1. run `yarn add @geistclone/core` or `npm i @geistclone/core` install it.

2. import into project:

```jsx
import { GeistProvider, CssBaseline } from '@geistclone/core'

const Application = () => (
  <GeistProvider>
    <CssBaseline /> // ---> Normalize styles
    <AppComponent /> // ---> Your App Component
  </GeistProvider>
)
```
